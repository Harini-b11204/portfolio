# Harini Baskaran - Portfolio Website

Welcome to my professional portfolio website! Built with React.js and Tailwind CSS, this portfolio showcases my skills, experience, and projects as a full-stack developer and data analyst.

## 🌟 Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Hero Section**: Eye-catching introduction with resume download option
- **About Section**: Comprehensive overview of my professional background and current role at UPS Logistics
- **Skills Section**: Organized display of technical and professional skills
- **Projects Section**: Showcase of professional experience and machine learning projects
- **Contact Section**: Easy ways to connect via social media and email
- **Smooth Navigation**: Fixed navbar with smooth scrolling between sections

## 🛠️ Tech Stack

- **Frontend**: React.js, Tailwind CSS
- **Icons**: Material-UI Icons

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Harini-b11204/portfolio.git

# Navigate to the project directory
cd portfolio

# Install dependencies
npm install
```

### Running the Application

```bash
npm start
```

Runs the app in development mode. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Building for Production

```bash
npm run build
```

Builds the app for production to the `build` folder.
